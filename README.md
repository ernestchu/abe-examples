# ABE Examples
Examples for attribute-based encryption written in C++, featuring cyphertext-policy ABE (CP-ABE) and key-policy ABE (KP-ABE).

## Dependencies

- [OpenABE](https://github.com/zeutro/openabe)

## Usage

```sh
make
./cpabe
./kpabe
```
